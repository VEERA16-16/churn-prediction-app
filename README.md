# 🚀 Customer Churn Prediction App

**End-to-end ML project** predicting telecom customer churn using Logistic Regression (**F1=0.61**).

[![Streamlit App](screenshots/low-risk.png)](https://github.com/VEERA16-16/churn-prediction-app)

## 📊 Model Performance
| Model | Accuracy | F1-Score | Recall |
|-------|----------|----------|--------|
| Logistic Regression | **80.1%** | **61.0%** | **58.6%** |
| Random Forest | 78.6% | 54.6% | 48.4% |
| Decision Tree | 76.8% | 52.9% | 49.2% |

## ✨ Features
- **21 telecom features** (tenure, contract, charges, services)
- **Preprocessing pipeline** (encoding, scaling)
- **Streamlit web app** for instant predictions
- **Business recommendations** (discounts/upsell)
